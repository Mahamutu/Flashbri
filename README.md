# Flashbri

**Flashbri** is a simple browser-based vocabulary learning app for studying words from your own files.

[Open Flashbri](https://mahamutu.github.io/Flashbri/)

---

## About

Flashbri lets you import your own vocabulary list and learn it in different ways.  
You can choose the question direction, switch between multiple choice and typing mode, review your mistakes, and export the words you still need to practice.

The app works directly in the browser and does not require an account, database, or installation.

---

## Features

- Import word lists from `.txt` or `.csv` files
- Study in both directions:
  - foreign language → Polish
  - Polish → foreign language
- Multiple choice mode
- Typing mode
- Change quiz mode during learning
- Change question direction during learning
- Optional automatic move to the next question
- Adjustable auto-next delay from 1 to 5 seconds
- Review only incorrectly answered words
- Export incorrect answers to a new file
- Save recently opened word sets
- Pin favorite word sets
- Remove word sets from the list
- Light mode and dark mode
- Purple accent color theme
- Green feedback for correct answers
- Red feedback for wrong answers
- Works as a simple static web app
